# 工具集
这里有很多实用的工具，由python编写，并且内置打包后的exe(或者bat)包，方便快捷。
## 目录结构
根目录下的文件夹都是工具文件夹，每个工具文件夹里都有code和exe两个文件夹。code里面是源代码，exe里面是打包后的exe文件。
以下为目录示例：
```
工具1/
    code/
        1.py
    exe/
        1.exe

工具2/
    code/
        1.py
    exe/
        1.exe

......
```
### 说明
批处理工具源代码知可执行文件一样，只提供一个exe文件夹。
